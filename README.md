##  System Architecture: Multi-Stream Pipeline

The system employs a **Hybrid Fusion** approach, moving beyond simple pixel analysis to look for mathematical and semantic fingerprints.

### 1. Preprocessing Stage
**Input Image** 
**Preprocessing Pipeline** * **RGB Channel:** Standard pixel data for spatial features.
* **Frequency Domain:** FFT (Fast Fourier Transform) / DCT (Discrete Cosine Transform).
* **Noise Residual:** Extraction of high-frequency noise patterns (SRM).

---

### 2. The Triple-Stream Encoder
The core of the detection logic is split into three specialized streams:

| Stream | Architecture Recommendation | Forensic Focus |
| :--- | :--- | :--- |
| **Spatial Stream** | `ConvNeXtV2` / `EfficientNetV2` | Texture anomalies and blurring. |
| **Frequency Stream** | `FFT` / `DCT` + `CNN` | Grid artifacts and upsampling patterns. |
| **Semantic Stream** | `ViT` / `Swin` / `DINOv2` | High-level logic (e.g., mismatched eyes). |

---

### 3. Decision & Output Heads
* **Cross-Attention Fusion Layer:** Dynamically weighs the importance of each stream based on the input's characteristics.
* **Forgery Localization Head:** Generates a **Segmentation Mask** to highlight specific manipulated regions.
* **Classification Head:** Final label: `Real` vs. `AI-Generated` vs. `Manipulated`.
* **Explainability Module:** Provides a confidence score and heatmaps for forensic transparency.

---

## 🔍 Key Forensic Indicators
The hybrid architecture outperforms traditional models by learning these specific "tells":

* **GAN/Diffusion Fingerprints:** Mathematical signatures unique to specific model architectures.
* **Abnormal Frequency Spectra:** Non-natural peaks in the frequency domain caused by neural upsamplers.
* **Lighting & Shadow Mismatch:** Inconsistencies in global illumination often missed by 2D generators.
* **Noise Pattern Anomalies:** Discrepancies between sensor-based noise and algorithmically generated noise.
* **Semantic Inconsistencies:** Logic-based errors such as floating objects or anatomical impossibilities.
* **Compression Artifacts:** How the AI "interprets" JPEG/PNG compression compared to a real sensor.

> **Note:** The integration of the **Semantic Stream (ViT)** is critical for identifying "Perfect Fakes" where low-level pixel artifacts have been successfully smoothed or post-processed.
